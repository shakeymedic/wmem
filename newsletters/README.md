# Newsletters landing pages

HTML versions of each issue, automatically generated by the pipeline.

- `em/issue-N.html` — EM Evidence Rundown
- `phem/issue-N.html` — PHEM Evidence Rundown
- `anaes/issue-N.html` — Anaesthetics & ICU Evidence Rundown

Each page includes the full newsletter content styled consistently with the main site, making it searchable on wmebem.com. The PDF remains the archive/print copy via Google Drive.
